## 算法思想

**贪心法**

## 思路分析:

**个子高的人是看不见个子矮的人的，即个子矮的人不会对个子高的人的站位产生任何影响，所以个子矮的人应该优先选位。**

**Step 1 :**

将整个待排序序列按照如下规则排序:

1. 身高高的在前，身高矮的在后；

2. 当身高相同时，由于第二个参数反映的是在当前成员之前:身高**大于等于**当前成员的身高的成员个数，故：

将第二个参数值小的放在前面，参数值更大的置后。

**Step 2 :**

采取"**插空**"的方式进行新队列的建立：

对于按 **Step 1** 中准则排序后的成员队列，依次参照每个成员的第二个参数的值，将各成员插入新的队列中，完成排序任务。

(i.e. 高个子先站好，矮个子再找到自己的位置插进去)

## AC代码如下:
```C++
#include <bits/stdc++.h>
using namespace std;
class Solution {
public:
    vector<vector<int>> reconstructQueue(vector<vector<int>>& people) {
        vector<vector<int>> res;
        sort(people.begin(),people.end(),[](const vector<int>&a,const vector<int>&b){
            return a[0]>b[0] || a[0]==b[0]&&a[1]<b[1];
        });
        for(const vector<int>& person:people)
        {
            res.insert(res.begin()+person[1],person);
        }

        return res;
    }
};
```
