## 算法思想

**双指针法**

## 思路分析:

这是归并排序的合并操作的源码，本题大可以直接调用<algorithm>库中的sort函数直接实现排序，但是那样违背了本题训练的初衷。

这里的合并操作的思路是：对于这两个有序的非递减子数组，额外创建一个新的数组，然后分别使用一个指针指向各子数组的第一个元素，将两者的元素大小比较后，将较小的放入新数组中，

此后，移动提供较小元素的那个数组的指针，使之指向下一个数组中的元素，继续进行比对。同理，对比后将较小的数加入新数组，然后移动提供较小数的数组的指针，直至其中一方率先遍历完子数组中的所有元素。

此时，只需要将剩下的未遍历完的有序剩余子数组直接添加到新数组末尾即可。

## AC代码如下:
```
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        vector<int> res;
        int ptr1 = 0;
        int ptr2 = 0;
        while(ptr1 < m && ptr2 < n)
        {
            if(nums1[ptr1]<=nums2[ptr2])
            {
                res.insert(res.begin()+ptr1+ptr2,nums1[ptr1]);
                ptr1++;
            }
            else
            {
                res.insert(res.begin()+ptr2+ptr1,nums2[ptr2]);
                ptr2++;
            }
        }
        while(ptr1<m)
        {
            res.insert(res.begin()+ptr2+ptr1,nums1[ptr1]);
            ptr1++;
        }
        while(ptr2<n)
        {
            res.insert(res.begin()+ptr1+ptr2,nums2[ptr2]);
            ptr2++;
        }
        nums1 = res;

    }
};
```
