## 算法思想

**贪心法**

## 思路说明:

考虑到可以在位置i处种花的条件为 **flowerbed[i-1]==0&&flowerbed[i]==0&&flowerbed[i+1]==0**

故在向量首位各插入一个0，专门用于处理首尾位置的可种花情况。

```C++
flowerbed.insert(flowerbed.begin(),0);
flowerbed.push_back(0);
```

此后遍历整个向量，如果检测到符合种花条件的位置i,则设置 **flowerbed[i] = 1**,然后将n--,直到遍历结束。

遍历结束后，若n<=0则说明已完成种花任务。

1. n<0 说明实际可种花的位置数多于要求种花的数目n;
2. n=0 说明实际可种花的位置数恰好等于要求种花的数目n，即刚好种下。


## 以下为AC代码:

```C++
class Solution {
public:
    bool canPlaceFlowers(vector<int>& flowerbed, int n) {

        flowerbed.insert(flowerbed.begin(),0);
        flowerbed.push_back(0);
        for(int i=1;i<flowerbed.size()-1;i++)
        {
            if(flowerbed[i-1]==0&&flowerbed[i]==0&&flowerbed[i+1]==0)
            {
                flowerbed[i] = 1;
                n--;
            }
        }
       return n<=0;
    }
};
```
