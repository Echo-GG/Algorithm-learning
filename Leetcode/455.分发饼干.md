## 算法思想

**贪心法**

每个局部的解都是最优的，并且全局结果是局部结果的简单求和，且局部结果互不相干。

因此局部最优的策略也同样是全局最优的策略。

## 思路说明

给剩余孩子里最小饥饿度的孩子分配最小的能饱腹的饼干。

## AC代码如下:

```C++
class Solution {
public:
    int findContentChildren(vector<int>& g, vector<int>& s) {
        sort(g.begin(),g.end());
        sort(s.begin(),s.end());
        int size_g = g.size();
        int size_s = s.size();
        int count = 0;
        int m = 0;
        int n = 0;
        while(m<size_g&&n<size_s)
        {
            if(g[m]<=s[n])
            {
                count++;
                m++;
            }
            n++;
        }
        return count;
    }
};
```
