## 算法思想
**贪心法**

## 思路说明:

直接去追踪变化多样的股票涨跌过程是困难的，只需意识到股票的涨跌反映在相邻的天数之间无非就是后一天相较于前一天的股票价值增益：

如果增益为正则在前一天买入在后一天卖出。

如果增益为负，则见逃本次交易，即在当前的这对日期不做任何操作，即增益为0，从而继续检查下一对日期，直至所有交易都被考虑。

## AC代码如下:
```C++
#include <bits/stdc++.h>
using namespace std;
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int res = 0;
        for(int i=1;i<prices.size();i++)
        {
            res+=max(0,prices[i]-prices[i-1]);
        }
        return res;
    }
};
```
