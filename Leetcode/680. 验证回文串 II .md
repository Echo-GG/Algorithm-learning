## 算法思想

**双指针法**

## 思路分析:

考虑编写一个判断给定区间内字符串是否为回文串的辅助函数 **isPalindrome(const string& s, int start, int end){}**

容易使用双指针的思想完成回文串的判断逻辑编写。

此后考虑本题的特殊限定，反映在代码上无非就是两种情况:

1. 删除左指针指向的字符 **[low + 1, high]** 区间内的字符串为回文串;

2. 删除右指针指向的字符 **[low ,high - 1]** 区间内的字符串为回文串。

两种情况任意成立一种返回值即为**true**,字符串本身不需经过删除一个字符的操作就为回文串的，也返回**true**。

本题完成。

## AC代码如下:
```C++
#include <bits/stdc++.h>
using namespace std;
class Solution {
public:
    bool isPalindrome(const string& s,int start,int end)
    {
        int n = s.length();
        int l = start;
        int r = end;
        while(l<r)
        {
            if(s[l]==s[r])
            {
                l++;
                r--;
            }
            if(s[l]!=s[r])
            {
                return false;
            }
        }
        return true;
    }

    bool validPalindrome(string s) {
        int n = s.length();
        int l = 0;
        int r = n-1;
        while(l<r)
        {
            if(s[l]==s[r])
            {
                l++;
                r--;
            }
            else
            {
                return (isPalindrome(s,l+1,r)||isPalindrome(s,l,r-1));
            }
        }

        return true;
    }
};
```
