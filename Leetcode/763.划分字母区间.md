## 算法思想

**贪心法**

## 思路分析

由于同一个字母只能出现在同一个片段，显然同一个字母的第一次出现的下标位置和最后一次出现的下标位置必须出现在同一个片段。

因此需要遍历字符串，得到每个字母最后一次出现的下标位置。

解决方法:

维护一个last[26]数组，用来记录遍历目标字符串时，当前字符串中字母的最后出现位置。

基本逻辑如下:
```C++
  int last[26];
  int len = s.length();
  for(int i=0;i<len;i++)
    last[s[i]-'a'] = i;
```

在得到了每个字母在字符串中的最后出现位置后，我们只需要再次遍历字符串，采用贪心的思想：

使用start和end两个指针，不断更新end指针使得 end = max(end,last[s[i]-'a']),

当遍历到更新后的end指针时:

只需再根据新的end指针更新start指针: start = end+1，即可在完成前一区间的划分后，从而继续开启下一区间的查找。

结果向量中记录各子区间长度等于end-start+1.

## AC代码如下:
``` C++
#include<bits/stdc++.h>
using namespace std;
class Solution {
public:
    vector<int> partitionLabels(string s) {
        int last[26];
        int len = s.length();
        for(int i=0;i<len;i++)
        {
            last[s[i]-'a'] = i;
        }

        int start = 0;
        int end = 0;
        vector<int> res;
        for(int i=0;i<len;i++)
        {
            end = max(end,last[s[i]-'a']);
            if(i==end)
            {
                res.push_back(end-start+1);
                start = end+1;
            }
        }
        return res;
    }
};
```
