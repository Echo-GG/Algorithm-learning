## 算法思想

**双指针法**

## 思路分析:

常规的双指针查找操作：

由于序列已按非递减顺序排列好：

1. 和小于目标值则右移左指针；

2. 和大于目标值则左移右指针；

3. 和等于目标值则记录此时的下标(1-based), 并退出循环。

## AC代码如下:

```C++
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<int> twoSum(vector<int>& numbers, int target) {
        int n = numbers.size();
        int l = 0;
        int r = n-1;
        int idx1 = -1;
        int idx2 = -1;
        while(l<r)
        {
            int sum = numbers[l] + numbers[r];
            if(sum == target)
            {
                idx1 = l+1;
                idx2 = r+1;
                break;
            }
            else if(sum < target)
            {
                l++;
            }
            else if(sum > target)
            {
                r--;
            }
        }

        vector<int> res(2,-1);
        res[0] = idx1;
        res[1] = idx2;
        return res;
    }
};
```
