## 算法思想

**双指针法**

## 思路分析:

考虑(int) sqrt(c),然后采用传统的双指针法穷举移动求解即可。

## AC代码如下
```c++
class Solution {
public:
    bool judgeSquareSum(int c) {
        long a = 0;
        long b = (int)sqrt(c);
        while(a<=b)
        {
            long sum = a*a + b*b;
            if(sum < c)
            {
                a++;
            }
            else if(sum > c)
            {
                b--;
            }
            else
                return true;
        }
        return false;
    }
};
```
